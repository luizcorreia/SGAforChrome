SGAforChrome
============

SGA for Chrome é uma extensão criada com intuito de melhorar a experiência no antigo Sistema da PUC Minas.


Como Surgiu
===========

Desde quando entrei para PUC, fiquei decepcionado com o SGA, sendo ela uma universidade grande e de renome nacional esperava mais do seu sistema de gestão acadêmico. Ele faz o que deve mas a experiência é horrivel.

Mais tarde, vi que o sistema se comportava ainda pior no Google Chrome. Foi então que pensei em criar um script para o greasemonkey. Pesquisei na internet se tinha greasemonkey pra Chrome e descobri que eu poderia fazer a mesma coisa e até mais com uma extensão nativa.
Então fiz uma extensão básica que injetava algumas linha de CSS para consertar o menu bugado.
